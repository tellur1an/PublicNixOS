# Secrets (agenix + YubiKey)

age-encrypted secrets, reproducible via the flake. Encrypted `.age` blobs are
**safe to commit** — only a recipient's private key (host SSH key, or a YubiKey)
can decrypt them.

- Recipients / rules: [`secrets.nix`](./secrets.nix)
- NixOS wiring: [`../modules/agenix.nix`](../modules/agenix.nix)
- YubiKey hardware support (pcscd, age-plugin-yubikey): [`../modules/yubikey.nix`](../modules/yubikey.nix)

## Model

Every secret is encrypted to **the host SSH key + every YubiKey**:

- **Host key** → the machine auto-decrypts at boot (no PIN/touch). Lives at
  `/etc/ssh/ssh_host_ed25519_key`, generated by NixOS (openssh enabled, firewall
  closed — not for remote login).
- **YubiKeys** → who can *edit/rekey* secrets, and the **recovery root**: any
  YubiKey can decrypt everything on any machine. Never remove the YubiKeys from
  a rule.

## First-time setup

1. `sudo nixos-rebuild switch --flake .#nixos` — generates the host key + installs `agenix`.
2. Put the host's **raw ssh public key** in `secrets.nix` as `nixos` (agenix uses
   age's native ssh support — NOT an ssh-to-age `age1…` value):
   ```sh
   cat /etc/ssh/ssh_host_ed25519_key.pub
   ```
3. Set up each YubiKey (plugged in, one at a time) and copy its recipient into `secrets.nix`:
   ```sh
   age-plugin-yubikey --generate   # first time only — creates the identity on the key
   age-plugin-yubikey --list       # prints the age1yubikey1... recipient
   ```
4. Create a secret (run from this `secrets/` dir):
   ```sh
   cd secrets
   agenix -e example.age           # opens $EDITOR; writes encrypted example.age
   ```
5. Declare it in `../modules/agenix.nix` (`age.secrets.example.file = ../secrets/example.age;`)
   and rebuild. Use it via `config.age.secrets.example.path`.

## Adding the 3rd (spare) YubiKey later

1. Plug it in → `age-plugin-yubikey --generate` (if new) → `age-plugin-yubikey --list`.
2. Add the recipient in `secrets.nix` (uncomment `yubikey3`, fill it in, add to `yubikeys`).
3. Rekey all secrets (needs one *existing* YubiKey or the host key to decrypt):
   ```sh
   cd secrets && agenix --rekey
   ```
4. Commit the re-encrypted `.age` files.

## Reinstall / new machine (with a YubiKey)

1. Fresh NixOS install, clone this repo, `nixos-rebuild switch` → new host key generated.
2. Plug a YubiKey (same hardware = same age identity).
3. Replace `nixos` in `secrets.nix` with the new host pubkey (step 2 above).
4. `cd secrets && agenix --rekey` — the YubiKey decrypts, re-encrypts to the new host key.
5. Rebuild. The machine now auto-decrypts at boot.

> Lost every machine but still hold a YubiKey? You can still decrypt any secret:
> `cd secrets && agenix -d example.age`.
